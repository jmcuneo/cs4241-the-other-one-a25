Audio Visualizer — Web Audio + Canvas (work in progress)

I built this during the in-class ICE window. It runs and meets the rubric.

How to use
  1. Click Choose audio and load song file (mp3/wav/etc.).
  2. Use the sliders to change bar count, sensitivity, hue, and trail.
  3. Switch Mode between Bars and Radial. (Dots mode coming soon — commented out.)

Under the hood: AudioContext → AnalyserNode (FFT bins) → draw on <canvas>.

WIP: mic input + a third “dots” mode are commented out due to time/hosting constraints.
